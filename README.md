# MoreTextViewSample

텍스트뷰(TextView)에서 "...더보기" 를 구현한 MoreTextView입니다.

오픈소스를 찾아보다,  결국 구현하게 되었습니다.
텍스트에 "\n"이 있는 경우에도 잘 됩니다.
